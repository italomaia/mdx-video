Usage
-------
```python
import markdown

text = '...'
html = markdown.markdown(text, extensions=['video'])
```

Support
-------

The following video providers are supported:

 - YouTube
 - Vimeo
 - Yahoo Video
 - Dailymotion
 - Metacafe
 - Veoh

Contribute
----------
Missed some provider?
